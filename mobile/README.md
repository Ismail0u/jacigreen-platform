# JACIGREEN Mobile Prototype

Ce dossier contient une application mobile Expo minimale pour le projet JACIGREEN.

## Installation

```bash
cd mobile
npm install
```

## Démarrage

```bash
cd mobile
npm start
```

## Notes

- Le backend est attendu sur `http://localhost:8000`.
- Si vous utilisez un émulateur Android, remplacez `localhost` par `10.0.2.2`.
- L'application affiche une liste de missions et les détails d'une mission sélectionnée.
